[**Agentica**](http://www.symbolica.ai/agentica) is an open-source agent framework. 
* Run agents with the [Python SDK](https://github.com/symbolica-ai/agentica-python-sdk) 
* ... or the [TypeScript SDK](https://github.com/symbolica-ai/agentica-typescript-sdk) 


**Need an API key?** Head over to the [Symbolica platform](https://www.symbolica.ai/agentica).
**Want to develop locally?** Run the [Agentica Server](https://github.com/symbolica-ai/agentica-server).
**Found a bug?**
File an issue in our [Issue Repo](https://github.com/symbolica-ai/agentica-issues).
