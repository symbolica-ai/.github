### Agentica is open source software.

- Run agents with [Agentica Server](https://github.com/symbolica-ai/agentica-server).
- Integrate them with the [Python SDK](https://github.com/symbolica-ai/agentica-python-sdk).
- Or orchestrate them with the [TypeScript SDK](https://github.com/symbolica-ai/agentica-typescript-sdk).
- Found a bug? File an issue in our [Issue Repo](https://github.com/symbolica-ai/agentica-issues).
